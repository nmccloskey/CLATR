# CLATR - Computational Linguistic Analysis of Text for Research

CLATR is a Python program that runs lexical, morphosyntactic, phonological, semantic, and clustering analyses on English text samples. The program is designed for language researchers interested in computational analysis and there are no installation or command line entries required - everything runs through Google Colab.

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
